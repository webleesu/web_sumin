#### 색상 표기

- RGB

  - `Red, Green, Blue`로 이루어져 있고 3개의 색을 모두 섞으면 흰색이 된다.
  - 주로 디지털과 웹에 이용된다.
  - webRGB는 HEX라고 불리우고 16진수로 표현된다.

- CMRK

  - `Cyan, Megenta, Yellow, Black`으로 이루어져있고 3개의 색을 모두 섞으면 검은색이 된다.
  - 주로 프린트에 이용된다.

- hsl `hue, saturation, lights` (hsb)`hue, saturation, brightness`

  - 색상, 채도, 명도로 이루어진 색 표현방법

  - 색상 - 0~360

    채도 0~100%

    명도 0~100%

    읽을땐 명도, 채도, 색상 순으로 읽음



#### 파일 저장 형식

- Illustrator

  - *.ai

    일러스트레이터 기본 파일 형식, 벡터방식 (2GB 이상 불가)

  - *.eps

    벡터방식, 인쇄목적 (ps파일)

  - *.pdf

    범용 파일저장방식, 보안성이 좋음

  - *.svg

    벡터형 웹그래픽 파일, 코딩가능

    

- Photoshop

  - *.psd

    포토샵 기본 파일 형식, (2GB 이상 불가)

  - *.jpg

    기본 이미지 저장 방식, 여러번 저장 할 경우 이미지 훼손

  - *.png

    웹용 이미지 저장방식 (8bit, 24bit 두가지 있음), RGB 형식

  - *.gif

    움직이는 이미지 저장에 사용하는 방식



- 기타

  - webp

    웹용 이미지 파일 (구글제작), 무료 라이센스

  - webm

    웹용 영상파일 (구글제작), 무료 라이센스

  - OGG

    영상, 음악, 무료 라이센스

#### figma

- 그래픽과 웹디자인에서는

  y 좌표는 아래로 내려갈 수록 -

  x 좌표는 그대로 오른쪽으로 갈 수록 +

- Clip content 에 체크 되어있으면 프레임에서 도형이 일부 나가면 가려짐

- 도형 크기 기우면서 shift 누르면 비율 유지

- `ctrl + G` : 그룹으로 묶기

  - 그룹이기 때문에 constraints 에서 그룹 내의 각각의 도형별로 기준이 잡힘

- `ctrl + alt + G` : 프레임으로 묶기

  - 프레임이기 때문에 프레임 내의 각각 도형은 프레임 사이즈에 기준이 잡힘
  - 프레임도 따로 기준이 잡힘

```markdown
그룹, 프레임을 나누는 기준

틀의 크기를 변경해야하는 경우

자신을 기준으로 처리한다면 **프레임**

자신의 내부의 요소들의 기준을 따라가는 경우 **그룹**

그런데 뭔지 뭔지 모르겠다면? 그냥 **프레임**
```


- `Tidy` : 전체를 봤을때 간격을 맞춰줌
- 프레임 푸는 법 : 대략적으로 `Ungroup (Ctrl+Shift+G)`

- `add outo layout` : 레이아웃을 자동적으로 잡아주며 새 도형을 합칠 수도 있다.
  - outo layout 안에 레이아웃을 또 넣을 수 있음
- 프레임을 다른 페이지로 이동 가능 : `move to page`
- `creat component (우클릭/가장 상단의 다이아몬드 모양)`  : 만들면 추후 복사한 모든 컴포넌트가 같이 변경됨.
- `export` : 원하는 객체 선택 후 export로 원하는 형식으로 파일 생성 가능(이미지 `png, jpg, pdf, svg`)
