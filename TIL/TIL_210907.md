#### figma

- 그래픽과 웹디자인에서는

  y 좌표는 아래로 내려갈 수록 -

  x 좌표는 그대로 오른쪽으로 갈 수록 +

- Clip content 에 체크 되어있으면 프레임에서 도형이 일부 나가면 가려짐
- 도형 크기 기우면서 shift 누르면 비율 유지
- `ctrl + G` : 그룹으로 묶기
  - 그룹이기 때문에 constraints 에서 그룹 내의 각각의 도형별로 기준이 잡힘
- `ctrl + alt + G` : 프레임으로 묶기
  - 프레임이기 때문에 프레임 내의 각각 도형은 프레임 사이즈에 기준이 잡힘
  - 프레임도 따로 기준이 잡힘

```markdown
그룹, 프레임을 나누는 기준

틀의 크기를 변경해야하는 경우

자신을 기준으로 처리한다면 **프레임**

자신의 내부의 요소들의 기준을 따라가는 경우 **그룹**

그런데 뭔지 뭔지 모르겠다면? 그냥 **프레임**
```


- `Tidy` : 전체를 봤을때 간격을 맞춰줌
- 프레임 푸는 법 : 대략적으로 `Ungroup (Ctrl+Shift+G)`

- `add outo layout` : 레이아웃을 자동적으로 잡아주며 새 도형을 합칠 수도 있다.
  - outo layout 안에 레이아웃을 또 넣을 수 있음
- 프레임을 다른 페이지로 이동 가능 : `move to page`
- `creat component (우클릭/가장 상단의 다이아몬드 모양)`  : 만들면 추후 복사한 모든 컴포넌트가 같이 변경됨.
- `export` : 원하는 객체 선택 후 export로 원하는 형식으로 파일 생성 가능(이미지 `png, jpg, pdf, svg`)
