#### position

- position
  - static : 자신의 위치를 지킴
  - realtive : 자신의 위치를 지키고 위치 설정해주면 현재 자신의 자리로부터 배치됨, (자기자신을 기준으로 맞춤)
  - absolute : 자신의 위치를 잃어버리고 위치 설정해준 곳으로 고정됨, **스크롤 따라가지 않음**
  - fixed : 자신의 위치를 잃어버리고 위치 설정해준 곳으로 고정됨, **스크롤 따라감**
- 위치 설정
  - top
  - left
  - right
  - bottom
  - z-index : 단위x

- html은 나중에 쓴 코드가 위에 배치됨
