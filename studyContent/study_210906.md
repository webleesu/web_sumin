#### 색상 표기

- RGB

  - `Red, Green, Blue`로 이루어져 있고 3개의 색을 모두 섞으면 흰색이 된다.
  - 주로 디지털과 웹에 이용된다.
  - webRGB는 HEX라고 불리우고 16진수로 표현된다.

- CMRK

  - `Cyan, Megenta, Yellow, Black`으로 이루어져있고 3개의 색을 모두 섞으면 검은색이 된다.
  - 주로 프린트에 이용된다.

- hsl `hue, saturation, lights` (hsb)`hue, saturation, brightness`

  - 색상, 채도, 명도로 이루어진 색 표현방법

  - 색상 - 0~360

    채도 0~100%

    명도 0~100%

    읽을땐 명도, 채도, 색상 순으로 읽음



#### 파일 저장 형식

- Illustrator

  - *.ai

    일러스트레이터 기본 파일 형식, 벡터방식 (2GB 이상 불가)

  - *.eps

    벡터방식, 인쇄목적 (ps파일)

  - *.pdf

    범용 파일저장방식, 보안성이 좋음

  - *.svg

    벡터형 웹그래픽 파일, 코딩가능

    

- Photoshop

  - *.psd

    포토샵 기본 파일 형식, (2GB 이상 불가)

  - *.jpg

    기본 이미지 저장 방식, 여러번 저장 할 경우 이미지 훼손

  - *.png

    웹용 이미지 저장방식 (8bit, 24bit 두가지 있음), RGB 형식

  - *.gif

    움직이는 이미지 저장에 사용하는 방식



- 기타

  - webp

    웹용 이미지 파일 (구글제작), 무료 라이센스

  - webm

    웹용 영상파일 (구글제작), 무료 라이센스

  - OGG

    영상, 음악, 무료 라이센스
