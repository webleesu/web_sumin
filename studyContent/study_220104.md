#### react

- `npm start`  : 리액트 시작

- `className` : 리액트에서는 class명을 `className`으로 써야함

- 리액트는 꼭 닫힌 코드가 있어야함

- 리액트는 불러올때 꼭 하나의 코드로 묶여있어야 한다.

  - 처음 생성되는 `.App`이 싫다면 `<></>` 이런 빈 코드로 사용해도 됨

- `{}` 로 묶으면 js코드 작성 가능

- return위에 변수 작성 후 해당 컴포넌트에 지정
